# sari-sari-utang-tracker
 Web-based utang tracking system for sari-sari stores using Flask and SQLite

## Render deployment

This repo includes `render.yaml` for a Render web service.

Render settings:

- Build command: `pip install -r requirements.txt`
- Start command: `gunicorn app:app`
- Health check path: `/healthz`
- SQLite database path: `/var/data/database.db`

The app uses a Render persistent disk mounted at `/var/data` so SQLite data survives restarts and deploys. Set `USER_NAME` and `PASSWORD` in the Render dashboard when prompted. `SECRET_KEY` is generated by Render.
